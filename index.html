How To Create a Personal Website on GitHub

by Luis Villa

First you will have to create a GitHub account by following these steps. After that you will have to signin using your new or existing Github account.

Once signed in follow the next steps:

  1. Click on "Repositories".
  2. Click on "New".
  3. Enter yourUserName.github.io (where yourUserName is the one you created).
  4. Click on "Create Repository".
  5. Click on "upload an existing file".
  6. Drag any HTML files or images. This is the content that will be dislayed on the website.
  7. Click on "Commit Changes".

You are basically done! Now open an browser and go to yourUserName.github.io to view your webpage.
